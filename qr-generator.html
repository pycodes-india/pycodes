About This QR Code Generator

This free tool creates scannable QR codes for links, plain text, Wi-Fi logins, email, phone numbers, and SMS. Everything runs locally in your browser — nothing you type is uploaded to a server. Customize the colors, size, and error correction, optionally add a center logo, then download the result as a PNG or SVG file.

How to Create a QR Code

  1. Pick a content type at the top: Link, Text, Wi‑Fi, Email, Phone, or SMS.
  2. Fill in the fields — for example, paste a URL or enter your Wi-Fi network name and password.
  3. Optionally adjust the foreground/background color, size, and error correction level.
  4. Optionally upload a logo to place in the center of the code.
  5. Watch the live preview update, then download as PNG or SVG, or copy the raw data string.

Frequently Asked Questions

What is a QR code and how does it work?

A QR code is a square pattern of black and white modules that stores data such as a link, text, or Wi-Fi details. Scanning it with a phone camera decodes that data instantly, without typing it manually.

Is this QR code generator free to use?

Yes, this tool is completely free with no signup, no watermark, and no limit on how many QR codes you can create.

Does my data get uploaded to a server when I generate a QR code?

No. The QR code is generated entirely in your browser using JavaScript. Your link, text, or Wi-Fi password is never sent to a server.

What types of QR codes can I create with this tool?

You can create QR codes for website links, plain text, Wi-Fi network login, email (with subject and body), phone numbers, and SMS messages with a preset text.

Can I customize the QR code color and add a logo?

Yes, you can set a custom foreground and background color, adjust the size and quiet zone, and optionally add a center logo image.

Will a QR code with a logo still scan correctly?

Yes, as long as you raise the error correction level to Q or H before adding a logo. Higher error correction lets the code stay scannable even when part of it is covered.
